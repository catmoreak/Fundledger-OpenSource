# Fundledger 

Fundledger is a open-source platform for managing projects, grants, and transaction ledgers. It features a complete system for both normal users and administrators.

## Features

### User Panel
- **Dashboard:** Overview of active grants, announcements, and recent transactions.
- **Grant Requests:** Submit requests for project/events funding.
- **Transactions:** Log and track personal transaction history.
- **QR Scanner:** In-app scanner to verify project tasks or transactions allowing financial transparency.
- **Announcements:** Check Fund updates and Announcements in realtime.

### Admin Panel
- **Dashboard:** System-wide metrics, stats, and transaction trends.
- **Projects & Grants:** Review, approve, or reject funding requests and manage active projects.
- **Announcements:** Publish updates and notifications to users.
- **Auditing:** Complete view of system-wide transactions.

---

## Tech Stack

- **Frontend:** React+ Vite, TypeScript
- **Styling & Icons:** CSS, Lucide React
- **Data Vizualization:** Recharts
- **Database & Auth:** Supabase (PostgreSQL)
